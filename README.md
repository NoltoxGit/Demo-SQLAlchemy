# SQLAlchemy Training

Ce dépôt contient un projet d'entraînement autour de **SQLAlchemy**, l'ORM Python utilisé pour interagir avec des bases de données relationnelles.

## Objectif

Apprendre à :

- définir des modèles SQLAlchemy,
- créer et gérer des sessions,
- exécuter des requêtes CRUD,
- structurer une application Python avec une base de données.

## Contenu

Le projet sert de support pratique pour tester des notions de modélisation, de persistance des données et d'utilisation d'un moteur de base de données avec Python.
